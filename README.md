# GitHubFollowers
Sean Allen's iOS Course - Take Home Project

This app allows you to see the list of followers of a GitHub profile you search for. 
- Open the profile details to learn about their Public Repos & Gists count, Followers & Following count. 
- Open GitHub Page on Safari by clicking 'GitHub Profile' button.
- Add your favorite followers to your favorites list. 
- Search for someone from your followers via the search bar.
